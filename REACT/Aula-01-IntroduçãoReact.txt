DEFINIÇÕES
  React: é uma bibliotaca de JavaScript sendo mais utilizada para aplicações web ela visa deixar a interface para o usuário mais fluída ao momento que o código é digamos compactado e reutilizado.

Biblioteca vs Framework:
  Bibliotecas são um lugar onde são armazeanadas diversas informações e dados criados trazendo para o desenvolvedor uma maior praticidade de elaboração de códigos.
  Framework são como biblioteca mas podendo serem criados a partir de uma seus códigos já estão concretizados, sua estrutura já está pronta para ser realizada.

Props: é a forma reduzida de propriedades e serve para fazer a comunicação dos dados entre os componentes, podemos dizer que é como se fossem os parâmetros de funções só que do React.

Hooks: são como ferramentas que permitem aos desenvolvedores organizar e gerenciar melhor as informações e interações na parte visual do que estão construindo. Tornam mais fácil lidar com coisas como atualizações de dados e ações do usuário, facilitando o processo de criação de páginas web.

Componentes: são funções que retornam JSX ou outros componentes, buscam deixar o código mais fluído e sem repetições desnecessárias
